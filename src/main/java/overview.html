Yarn Inventory Management

This project is a Java-based application for managing a personal yarn inventory. It helps users add, remove, filter, and merge yarns, estimate yarn usage for projects, and store past projects for future reference.

Features

  • Manage yarn inventory (add, remove, update, merge)
  • Filter and search yarn by color, brand, and weight
  • Estimate yarn usage for various projects
  • Save and load project plans
  • Generate detailed project reports

Project Structure

The application consists of several modules:

  • app - Contains the main entry point
  • utils - Utility classes for validation and calculations
  • utils.inventory - Handles yarn inventory operations
  • utils.project - Manages projects and estimates yarn usage
  • utils.yarn - Defines yarn properties and weights
